# Simple-360-Panorama-Viewer

ブラウザだけで動作する、シンプルな360度パノラマ画像ビューアです。

正距円筒図法（equirectangular）の360度画像を読み込み、PC・スマートフォンのどちらでも直感的に閲覧できます。画像はローカル環境のブラウザ内で処理され、サーバーには送信されません。

## Features

- 360度パノラマ画像の表示
- 正距円筒図法（横:縦 = 2:1）の画像に対応
- ローカル画像ファイルの読み込み
- ドラッグ＆ドロップでの画像読み込み
- マウスドラッグ / タッチ操作による視点移動
- ホイール / ピンチ操作によるズーム
- フルスクリーン表示
- 表示リセット機能
- レスポンシブ対応
- サーバー不要の静的HTMLアプリ

## Demo

GitHub Pages で公開した[デモページ](https://mssmat.github.io/Simple-360-Panorama-Viewer/)からすぐに試せます。

ページを開き、「画像を選択」からローカルの360度画像を読み込んでください。

読み込んだ画像はブラウザ内で処理され、サーバーには送信されません。

ローカルで試す場合は、`index.html` をブラウザで開いて利用できます。

## Usage

### 1. ファイルを開く

このリポジトリをクローン、またはダウンロードします。

`index.html` をブラウザで開きます。

### 2. 画像を読み込む

画面上の「画像を選択」ボタンから、360度パノラマ画像を選択します。

対応画像の目安:

- JPG
- PNG
- WebP
- その他ブラウザが対応している画像形式

推奨される画像比率:

```text
横:縦 = 2:1
```

例:

```text
4096 x 2048
8192 x 4096
```

## Controls

### PC

| 操作 | 内容 |
|---|---|
| ドラッグ | 視点を回転 |
| マウスホイール | ズームイン / ズームアウト |
| ↺ ボタン | 表示をリセット |
| ⛶ ボタン | フルスクリーン切り替え |

### Smartphone / Tablet

| 操作 | 内容 |
|---|---|
| 1本指ドラッグ | 視点を回転 |
| 2本指ピンチ | ズームイン / ズームアウト |
| ↺ ボタン | 表示をリセット |
| ⛶ ボタン | フルスクリーン切り替え |

## Privacy

読み込んだ画像はブラウザ内で処理されます。

画像ファイルはサーバーへアップロードされません。  
ローカル環境で安全に360度画像を確認できます。

## Notes

360度画像は、正距円筒図法の画像を想定しています。

画像比率が 2:1 から大きく外れている場合でも読み込みは可能ですが、表示が歪むことがあります。その場合、アプリ上に警告メッセージが表示されます。

## Customize

主な設定値は `script` 内の以下の定数で変更できます。

```js
const DEFAULT_FOV = 75;
const MIN_FOV = 30;
const MAX_FOV = 100;
const WHEEL_ZOOM_SPEED = 0.04;
const PINCH_SENSITIVITY = 0.06;
```

| 定数 | 内容 |
|---|---|
| `DEFAULT_FOV` | 初期視野角 |
| `MIN_FOV` | 最大ズーム時の視野角 |
| `MAX_FOV` | 最小ズーム時の視野角 |
| `WHEEL_ZOOM_SPEED` | マウスホイールのズーム感度 |
| `PINCH_SENSITIVITY` | ピンチ操作のズーム感度 |
